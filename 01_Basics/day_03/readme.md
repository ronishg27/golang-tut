# GoLang Basics - Day 03

## Topics Covered

- Conditional Statements
- Loops
- Functions

## Conditional Statements

- if else statement

```go
if (condition) {
	statement if true
} else {
	statement if false
}
```

## Loops

- for loop

```go
for [initialize]; [condition]; [increment/decrement] {
	statement
}
```
